WebQuery Features

  • Report freely on the data in your database
  • Access all fields (except overlap and binary) from tables related-to from the report's main table
  • Programmatically exclude specified tables and columns you do not wish users to have access to
  • Create calculated columns based on expressions
  • Spread report rows over multiple output lines
  • Add totals to numeric columns
  • Filter report data on the values in any field by a range of operators
  • Filter data with an expression
  • Order by any index or define your own ad hoc ordering
  • Define multi-level breaks to produce subheaders and subtotals
  • Build reports users can run directly from menus or elsewhere in your application
  • Output reports to the browser in a pop-up window, a new browser tab or a new browser window
  • Download reports as HTML, text, CSV or XML
  • Select font typeface and size for your report
  • Select any colour for text, background and alternate row highlighting
  • Include HTML in your reports to produce effects like hyperlinks and images (must be enabled)
  • Save queries either as public or (if logged in) private
